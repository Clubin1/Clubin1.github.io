Page Design

  • Home Page
  • Purpose: Gives a basic overview on MPA
  • Audience: Other Psychologists or people seeking mental help
  • Content: Texts and images
  • Data Entry: No data entry
  • Hyperlinks/dropdowns: Yes, there will be links
  • Actions: The only actions will be to take you to the professionals page
  • About Page
  • Purpose: Provides more in-depth information on MPAs cause
  • Audience: People seeking to learn more
  • Content: Paragraphs as well as a list of all the board members
  • Data Entry: No data entry
  • Hyperlinks/dropdowns: No, there will not be links
  • Actions: No certain actions
  • Contact Page
  • Purpose: Provide a form of communication to MPA
  • Audience: People that have any questions
  • Content: Just a contact form with entry fields
  • Data Entry: Yes, user will be asked for a name, email address and the message
  • Hyperlinks/dropdowns: The submit button will link to a mailto: hyperlink
  • Actions: Form submit/send email
  • Professionals Page
  • Purpose: Have a gallery of the members that are associated with MPA. Also gives upcoming events with MPA
  • Audience: People that wish to learn more
  • Content: Gallery of members and a list of events
  • Data Entry: No data entry
  • Hyperlinks/dropdowns: No, there will not be links
  • Actions: No certain actions
  • Application Page
  • Purpose: Have an application form for those wishing to be an MPA member
  • Audience: Other Psychologists
  • Content: An application form, in website and pdf
  • Data Entry: User can enter their personal information to apply
  • Hyperlinks/dropdowns: Hyperlink linking to an external pdf file
  • Actions: Submit form
  • Payment Page
  • Purpose: Allow members signing up to purchase a subscription.
  • Audience: Other Psychologists
  • Content: An embed of paypal to pay the costs
  • Data Entry: No Data entry
  • Hyperlinks/dropdowns: Button that opens up Paypal payment
  • Actions: Purchase subscription
